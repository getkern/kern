# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), v2.1.

We pledge to make participation a harassment-free experience for everyone. Examples of
unacceptable behaviour include harassment, trolling, insulting/derogatory comments, and
publishing others' private information.

Project maintainers are responsible for clarifying standards and may remove, edit, or reject
contributions, comments, commits, and other interactions that violate this Code of Conduct.

Instances of abusive or unacceptable behaviour may be reported to the maintainer via the
contact on the GitHub profile. All complaints will be reviewed and investigated promptly and
fairly.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
