kern

A fast, lightweight sandbox & virtual resource manager. early — but it runs

~5.5 ms to a rootless, kernel-enforced box — vs ~308 ms for docker run. Daemonless, no root, no runtime to install. Runs on Linux, Windows (via WSL2), and ARM boards — Raspberry Pi, NVIDIA Jetson, Arduino UNO Q — the same ~1 MB binary everywhere.

$ curl -fsSL https://raw.githubusercontent.com/getkern/kern/main/install.sh | sh

Linux · macOS (WSL) · ARM boards (Pi / Jetson / Arduino UNO Q) — checksum-verified static binary, no toolchain.

PS> irm https://raw.githubusercontent.com/getkern/kern/main/install.ps1 | iex

Windows — one line in PowerShell: sets up WSL2 + kern automatically, no Docker Desktop. Inspect first: install.sh · install.ps1.

$ scp kern <board>: && ssh <board> kern box edge --image alpine -- sh

ARM boards (Raspberry Pi, NVIDIA Jetson, Arduino UNO Q) — one ~1 MB static binary you scp and run: no daemon, no package. Or run the one-liner above on the board (it auto-detects aarch64). Runs where Docker won't even install — see EDGE.md.
